优美的Sublime Text

Jun 07, 2021杂项点击

Sublime Text 是一个轻量、简洁、高效、跨平台的编辑器。

Sublime 的下载与安装

直接去sublime的官网：http://www.sublimetext.com/ 下载Sublime Text 4，或者https://www.423down.com/4966.html这个网站上下载。

然后通过下载得到的.exe文件进行安装，可以自行选择安装的路径，安装过程很简单，直接按照默认选项进行安装即可，其中注意如下图：（记得勾选Add to explorer context menu,这样在右键单击文件时就可以直接使用Sublime Text打开。）

sublime本身是免费使用的，但不购买的话标题栏上会出现UNREGISTERD，你也可以使用破解的方法消除它，如 完美永久破解最新Sublime Text 4 Build 4107的方法：

  • 使用浏览器打开网站：https://hexed.it/
  • 在网站中打开Sublime Text安装目录并选择文件：sublime_text.exe
  • 搜索80 38 00并更改为FE 00 90（第一个匹配）
  • 另存文件到本地，即备份sublime_text.exe文件（只需重命名）
  • 将修改过的sublime_text.exe复制到sublime text 4安装目录中

修改侧边栏字体大小

参考：https://www.jianshu.com/p/906c48f2da5c，需要安装 **PackageResourceViewer **插件。

  1. 修改侧边栏字体大小(对应主题)

    打开 Command Palette （快捷键 Ctrl+Shift+P ）

    输入 PackageResourceViewer: Open Resource 回车
    在包列表中输入 Theme - Spacegray 回车
    再输入Spacegray.sublime-theme 回车
    在打开的文件中搜索 “class”: “sidebar_label”
    在 “class”: “sidebar_label” 后面追加如下代码：”font.size”:16, // 将字体大小设置为16。
    保存，修改完成。

  2. 修改行间距

    在Spacegray.sublimt-theme 中搜索”class”: “sidebar_tree”

    根据需要调整”row_padding”: [8, 8], 的值即可。

快捷键

使用快捷键可以显著提高程序员的开发效率，所以是有必要掌握的。

  1. 搜索文件：ctrl+p 输入文件名
  2. 搜索函数/方法：ctrl+p 输入文件名@方法名 如 Process@show_img
  3. 跳转到指定行：ctrl+p 输入文件名:行号，只输入:时在当前文件跳转
  4. 查找当前文件方法：ctrl+r
  5. 查找当前项目文件中的所有方法：ctrl+shift+r
  6. 返回/前进编辑位置：alt + -、alt + shift + -
  7. 切换标签页：ctrl+PgUp、ctrl+PgDp
  8. 选中单词：ctrl+d 连续按会选中页面中所有单词，以实现批量编辑
  9. 以单词为单位快速移动光标：ctrl + ⬅、ctrl + ➡
  10. 选中当前行：ctrl+L
  11. Ctrl+Shift+L：选择多行
  12. 跳转到第几行：ctrl+g
  13. Ctrl+W：关闭当前打开文件
  14. Ctrl+Shift+W：关闭所有打开文件
  15. 跳转到对应括号：ctrl+m
  16. 开关侧栏：ctrl+K+B
  17. 选中当前括号内容，重复(如按两次) 可选着括号本身：ctrl+shift+m
  18. 注释当前html标签块：ctrl+shift+/
  19. 专注编写模式：shift+F11
  20. 分屏显示：atl+shift+数字
  21. ctrl+Enter 在下一行插入新行，即使光标不在行尾也能快速向下插入一个空行
  22. ctrl+Shift+Enter 在上一行插入新行，即使光标不在行首也能快速向上插入一个空行
  23. ctrl+Shift+[ 选中代码，按下快捷键，折叠代码
  24. ctrl+Shift+] 选中代码，按下快捷键，展开代码
